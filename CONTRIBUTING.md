# Contributing to Awesome MoltBook

Thanks for your interest in contributing! This list is community-driven.

## How to Contribute

### Adding a Project

1. **Fork** this repository
2. **Add** your link to the appropriate section in README.md
3. **Follow** this format:
   ```markdown
   - **[Project Name](https://link.com)** - Brief description. ([Docs](https://docs.link.com))
   ```
4. **Submit** a Pull Request with:
   - Title: `Add [Project Name]`
   - Description: Why this project belongs here

### Quality Guidelines

**Projects should be:**
- ✅ Actively maintained (commits in last 3 months)
- ✅ Actually functional (not vaporware)
- ✅ Useful to the MoltBook community
- ✅ Safe (no credential harvesting, malware, scams)

**We do NOT list:**
- ❌ Tokens without utility
- ❌ Manifestos without code
- ❌ Abandoned projects
- ❌ Known scams or phishing

### Reporting Scams

If you discover a scam, phishing site, or malicious project:

1. **Open an Issue** with `[SECURITY]` prefix
2. Include:
   - Name/URL of the threat
   - Type (phishing, credential harvesting, scam, etc.)
   - Evidence (screenshots, links)
3. We will add it to the Red Flags section

### Updating Information

If a project URL changes, gets abandoned, or information becomes outdated:

1. Open an Issue or PR
2. Explain what changed
3. Provide new/correct information

## Code of Conduct

- Be respectful
- Focus on facts, not drama
- Help build something useful
