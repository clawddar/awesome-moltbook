## Relevant for MoltBook Ecosystem

### Multi-Agent Frameworks
- **AutoGen** (Microsoft) - Multi-agent conversation framework
  - GitHub: https://github.com/microsoft/autogen
  - Paper: https://arxiv.org/pdf/2308.08155.pdf
  - Discord: https://discord.gg/pAbnFJrkgZ
  
- **AgentVerse** (OpenBMB) - Task-solving & simulation agents
  - GitHub: https://github.com/OpenBMB/AgentVerse
  - Paper: https://arxiv.org/abs/2308.10848
  
- **Agents** (AIWaves) - Library for building language agents
  - GitHub: https://github.com/aiwaves-cn/agents
  - Features: Long-short term memory, tool usage, web navigation, multi-agent comms

### Coding Agents
- **Aider** - Pair programming with GPT in terminal
  - GitHub: https://github.com/paul-gauthier/aider
  - Site: https://aider.chat/
  - Works with existing codebases

### General Purpose
- **AutoGPT** - 140k+ stars, fully autonomous GPT-4
  - Site: https://agpt.co/
  - Internet access, memory management, file operations, plugins

- **AgentGPT** - Browser-based no-code AutoGPT
  - Site: https://agentgpt.reworkd.ai/
  - GitHub: https://github.com/reworkd/AgentGPT

- **AgentPilot** - Desktop app for building/managing agents
  - GitHub: https://github.com/jbexta/AgentPilot
  - Integrated with Open Interpreter and MemGPT

- **AgentForge** - Low-code agent building platform
  - GitHub: https://github.com/DataBassGit/AgentForge
  - Supports multiple LLMs (OpenAI, Claude, local models)

### Specialized
- **Adala** - Autonomous data labeling agent
  - GitHub: https://github.com/HumanSignal/Adala
  - Reliable, controllable, data processing focused

- **Agent4Rec** - Recommender system simulator (1,000 agents)
  - GitHub: https://github.com/LehengTHU/Agent4Rec
  - Uses MovieLens-1M dataset

- **AIlice** - Task planning with agent-calling trees
  - GitHub: https://github.com/myshell-ai/AIlice
  - Auto-creates agent trees for complex tasks
